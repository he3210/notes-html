1 镜像常用命令

$ sudo docker images                        # 列出本地 docker 镜像
$ sudo docker rmi he3210/apache2:webserver  # 删除本地镜像
$ sudo docker history e78b7f1e4671          # 查看镜像构建过程

2 Docker Hub 操作

$ sudo docker login                # 登录
$ sudo docker logout               # 登出
$ sudo docker pull ubuntu:12.04    # 从 Docker Hub 拉取镜像
$ sudo docker search puppet        # 在 Docker Hub 搜索镜像

4 提交定制容器进行创建镜像

$ sudo docker commit -m"ubuntu apache2" -a"shizhonghe" bdd311e829ad he3210/apache2:webserver
  • -m 指定新创建的镜像的提交信息
  • -a 新镜像作者
  • bdd311e829ad 指定想要制作成镜像的容器 ID
  • he3210/apache2:webserver 用户名/仓库名:标签

5 查看指定容器的端口被映射到哪了

$ sudo docker port static_web 80
0.0.0.0:32769

可看出，static_web 容器的 80 端口被映射到宿主机的 32769 端口了。当然，也可通过 $ docker ps 命令查看

6 docker volume

$ docker volume ls                # 列出所有volume
$ docker volume rm f5f4ff5a9      # 删除指定volume
$ docker volume prune             # 删除所有没用的volume
$ docker volume inspect f5f4ff5a9 # 显示volume详情
$ docker volume create my-vol     # 创建数据卷my-vol
