1 在 command group 中的区别

() 将 command group 置于 sub-shell 中去执行，也称 nested sub-shell
{} 则是在同一个 shell 内完成，也称 non-named command group。括号内变量可以在后续括号外使用

书写格式： 两者只要括号和命令之间有个空格，并且命令之间用分号隔开就不会出错

2 command meta 的区别

$() 命令替换(command substitution)，类似于 `cmd`

${} 变量替换。例如: $ A=B; echo ${A}B

shell 字符串的非贪婪(最小匹配)左删除
$ file=/dir1/dir2/dir3/my.file.txt
$ echo ${file#*/} # 从变量 file 左边开始删除，一直删除到第一个 / 右面
dir1/dir2/dir3/my.file.txt
$ echo ${file#*.} # 从变量 file 左边开始删除，一直删除到第一个 . 右面
file.txt

shell 字符串的贪婪(最大匹配)左删除
$ echo ${file##*/} # 从变量 file 左边开始删除，一直删除到最后一个 / 右面
my.file.txt
$ echo ${file##*.} # 从变量 file 左边开始删除，一直删除到最后一个 . 右面
txt

shell 字符串的非贪婪(最小匹配)右删除
$ echo ${file%/*} # 从变量 file 右边开始删除，一直删除到第一个 / 左面
/dir1/dir2/dir3
$ echo ${file%.*} # 从变量 file 右边开始删除，一直删除到第一个 . 左面
/dir1/dir2/dir3/my.file

shell 字符串的贪婪(最大匹配)右删除
$ echo ${file%%/*} # 从变量 file 右边开始删除，一直删除到最后一个 / 左面

$ echo ${file%%.*} # 从变量 file 右边开始删除，一直删除到最后一个 . 左面
/dir1/dir2/dir3/my

shell 根据下标提取字符串
$ echo ${1:4} # 从变量 file 中提取字符串，开始下标为 1，提取字符数量为 4
dir1
$ echo ${6:4} # 从变量 file 中提取字符串，开始下标为 6，提取字符数量为 4
dir2

shell 字符串变量值的替换
$ echo ${file/dir/path} # 把变量 file 中的第一个子串 dir 替换成 path
/path1/dir2/dir3/my.file.txt
$ echo ${file//dir/path} # 把变量中的所有子串 dir 替换成 path
/path1/path2/path3/my.file.txt

shell 字符串变量默认值
未定义 – unset
空 – null
非空 – non-null
$ echo ${file-file.txt} # 如果变量 file 未定义，返回 file.txt；否则，返回 ${file}
$ echo ${-file.txt} # 如果变量 file 未定义或为空，返回 file.txt；否则，返回 ${file}
$ echo ${file+file.txt} # 如果变量 file 已定义，返回 file.txt；否则，返回 ${file}
$ echo ${+file.txt} # 如果变量 file 非空，返回 file.txt;否则，返回 ${file}
$ echo ${file=file.txt} # 如果变量 file 未定义，返回 file.txt，同时给变量 file 赋值；否则，返回 ${file}
$ echo ${=file.txt} # 如果变量 file 未定义或为空，返回 file.txt，同时给变量 file 赋值；否则，返回 ${file}
$ file2=${file?file.txt} 2>err.log # 如果变量 file 未定义，输出 file.txt 到 STDERR；否则，返回 ${file}
$ file2=${?file.txt} 2>err.log # 如果变量 file 未定义或为空，输出 file.txt 到 STDERR；否则，返回 ${file}

计算 shell 字符串变量的长度
$ echo ${#file}

shell 数组
$ A=(a b c def) # 定义数组
$ echo ${A[@]} # 输出数组全部元素
a b c def
$ echo ${A[*]} # 输出数组全部元素
a b c def
$ echo ${A[0]} # 输出数组第一个元素
a
$ echo ${#A[@]} # 输出数组 length
4
$ echo ${#A[*]} # 输出数组 length
4
$ echo ${#A[3]} # 输出数组第4个元素的 length
3
$ A[2]=xyz # 数组元素重新赋值

$(()) 用来做整数运算。不能用于浮点数，因为它会当成字符串
bash 中支持的运算符有 + - * / % & | ^(异或 XOR) !
常用于(())的测试符号有 < > <= >= = !

$ a=11; b=2; c=4;
$ echo \((((a+b) * 2)) # 只要括号中的运算符、表达式符合C语言运算规则，都可用在\)((exp))
$ echo $((16#11)) # 输出十六进制的11对应的十进制
17
$ ((a++)) # 使用 (()) 重定义变量值
$ ((a<b)) # 如果 a<b，返回0（true）

3 其它

() 定义数组
array=(a b c d)

{} 花括号扩展。可以使用通配符对文件名进行扩展。花括号中不能有空白，除非这个空白被转义
例如：
$ ls {ex1,ex2}.sh
ex1.sh ex2.sh
$ touch {a..d}.sh
a.sh b.sh c.sh d.sh
$ ls {ex{1..3},ex4}.sh
ex1.sh ex2.sh ex3.sh ex4.sh
$ ls {ex[1-3],ex4}.sh
ex1.sh ex2.sh ex3.sh ex4.sh
